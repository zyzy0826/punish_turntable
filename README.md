# PUNISH//WHEEL

賽博龐克風格的懲罰轉盤網頁，適合派對、團隊活動等場合使用。

## 預覽

- 霓虹發光轉盤，Cyberpunk 視覺風格
- 點擊按鈕旋轉，停止後顯示懲罰結果
- 懲罰內容獨立存放，方便隨時新增或修改

## 使用方式

直接用瀏覽器開啟 `index.html`，無需安裝任何套件或啟動伺服器。

## 檔案結構

```
punish_turntable/
├── index.html        # 主頁面（轉盤畫面與動畫邏輯）
└── punishments.js    # 懲罰清單（內容與顏色）
```

## 新增 / 修改懲罰

編輯 `punishments.js`，照格式增減項目即可：

```js
const PUNISHMENTS = [
  { label: "你的懲罰文字", color: "#霓虹色碼" },
  // ...
];
```

- `label`：顯示在轉盤與結果彈窗上的文字
- `color`：該扇形的霓虹主色（建議使用飽和度高的顏色）

項目數量不限，轉盤會自動均分扇形。
